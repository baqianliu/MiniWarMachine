## 目录结构
* DOC 存放相关文档

* Source  存放源文件

* Tools 存放过程中使用的工具，一般是存放此部分使用的独有的工具，如果有共用的工具，该工具将存放至更高级别的目录

## 工程的创建与运行

工程由`CubeMX`创建，在`keil5MDK`中进行开发，目前使用串口进行烧录
